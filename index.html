projects

Songsnap

Songsnap is a personalized music app concept. Photograph yourself and grab a track from your Soundcloud favorites and playlists that matches your mood. Songsnap requires a Soundcloud account to try.

  • Built on Rails and React.js with Soundcloud authentication and music caching layer.
  • HTML5 video capture combined with Microsoft emotion API quantifies facial emotion.
  • Soundcloud API filters and returns a track from user's Soundcloud favorites and playlists.

Lyfecoach

Lyfecoach is a full-stack web app / social media platform designed for event-browsing and for following event hosts. Users have one place to stay updated from all of their favorite event organizers.

  • Social media app built on Rails and React.js with Flux architecture.
  • Query filtering and Google Geocoding API for an intuitive event browsing experience.
  • User authentication with BCrypt encryption.
  • Single page implementation of event and user browsing.
