projects

Lyfecoach

Lyfecoach is a full-stack event-browsing app inspired by EventBrite, but designed for following event hosts. Users have one place to stay updated from all of their favorite event organizers.

  • Social media app built on Rails and React.
  • Google Maps and Google Places API for searching nearby events.
  • User authentication with BCrypt encryption.
  • Single page implementation of event and user browsing.

'Sad Machine'

Maneuver between the soundwaves using the left and right arrow keys to move clockwise and counterclockwise. Try and beat your high score or relax and enjoy because you can never lose.

  • JavaScript game using HTML5's Canvas.
  • Simple but immersive design.
